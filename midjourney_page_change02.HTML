Commands

/imagine - Create images based on your text prompt.
/settings - Adjust version, mode, privacy, and generation speed.
/info - View your subscription info.
/subscribe - Select and pay for a suitable plan.
/blend - Combine multiple images into one.
/prefer option set - Save frequently used settings.
/prefer option list - View all saved settings for quick switching.
/help - Get tips and guidance for using the bot.

Shortcuts

--ar - Set image aspect ratio.
--q - Set image quality (0.25–5).
--no - Exclude details, e.g., "--no dog".
--uplight - Softer upscaling for minimal change.
:: - Adjust importance of prompt parts (-1 to 2).
--iw - Set image weight relative to text.
--chaos - Control randomness/diversity (0–100).
--seed - Generate repeatable results (0–4294967295).
--s - Set stylization strength (higher = more stylized).
